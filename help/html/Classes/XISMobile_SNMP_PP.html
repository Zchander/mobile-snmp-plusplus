Overview

Main class, containing three common commandline commands as methods

This class implements three common commands, so these are available to the programmer in Objective-C or Objective-C++

The project is based on SNMP++v3.2.25 from Jochen Katz, Frank Fock

discoverAgents:snmpVersion:remotePort:withCommunity:retry:timeout:error:

command: snmpDiscover – Returns an array with found SNMP agents on the specified network address

- (NSArray *)discoverAgents:(NSString *)onBroadcast snmpVersion:(uint)version remotePort:(NSNumber *)aPort withCommunity:(NSString *)community retry:(uint)retries timeout:(uint)timeout error:(NSError *__autoreleasing *)error

Parameters

onBroadcast

The broadcast address of the network you wish to scan

version

The SNMP version you wish to use

  1. SNMP version 1
  2. SNMP Version 2c
aPort

The UDP port used for the SNMP agent(s), defaults to 161

community

The readonly community string to identify yourself to the agent(s)

retries

The number of retries before returning the results. Maximum value is 100

timeout

The timeout in msec. Valid values are between 100 and 500 msec

error

Is an autoreleased NSError

Return Value

Returns an array containing the IP addresses of discovered SNMP agents

Discussion

command: snmpDiscover – Returns an array with found SNMP agents on the specified network address

walkOid:address:snmpVersion:remotePort:withCommunity:retry:timeout:walkSubTreeOnly:error:

Converted command: snmpGet – Returns the value corresponding to the requested OID

- (NSDictionary *)walkOid:(NSString *)oid address:(NSString *)hostAddress snmpVersion:(uint)version remotePort:(NSNumber *)aPort withCommunity:(NSString *)community retry:(uint)retries timeout:(uint)timeout walkSubTreeOnly:(BOOL)subTree error:(NSError *__autoreleasing *)error

Parameters

oid

The top level OID you wish to start the request

hostAddress

The address of the host you wish to query, might be an IPv4 address or a hostname

version

The SNMP version you wish to use

  1. SNMP version 1
  2. SNMP Version 2c
aPort

The UDP port used for the SNMP agent(s), defaults to 161

community

The readonly community string to identify yourself to the agent(s)

retries

The number of retries before returning the results. Maximum value is 100

timeout

The timeout in msec. Valid values are between 100 and 500 msec

subTree

A boolean which indicates wether you wish to walk only the subtree, or the whole tree. YES means you wish to walk only the subtree. NO means you want to read the whole SNMP tree.

error

Is an autoreleased NSError

Return Value

Returns a NSDictionary containing the OIDs and the corresponding values as NSString

Discussion

Converted command: snmpGet – Returns the value corresponding to the requested OID
